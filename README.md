# APEX — Rooftop Restaurant Website (Only Desktop Mode)

A 4-page website for APEX, a fictional rooftop restaurant 
in Ahmedabad. Built with pure HTML and CSS — no JavaScript, 
no frameworks.

## Pages
- **Home** — Hero section with background image and overlay
- **About** — Two-column layout with image and story
- **Reserve** — Reservation form with date, time, guest inputs
- **Contact** — Contact info and message form

## What I learned
- Multi-page navigation with working links
- Full-screen hero with background-image and rgba overlay
- CSS Flexbox for two-column layouts
- Form styling — date picker, time picker, select dropdown
- CSS hover transitions and animations
- Google Fonts — Playfair Display + Raleway
- Fixed navbar with scroll behavior
- Responsive layout basics

## Tech Stack
- HTML5
- CSS3
- Google Fonts
